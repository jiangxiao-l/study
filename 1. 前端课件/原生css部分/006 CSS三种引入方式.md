# CSS三种引入方式

## 一、三种方式的书写规范

#### 1、行间式

```html
<div style="width: 100px; height: 100px; background-color: red"></div>
```

#### 2、内联式

```html
<head>
    <style>
        div {
            width: 100px;
            height: 100px;
            background-color: red;
        }
    </style>
</head>
```

#### 3、外联式

```css
file: zero.css
div {
	width: 100px;
    height: 100px;
    background-color: red;
}
```

```css
file: zero.html
<head>
    <link rel="stylesheet" type="text/css" href="css/zero.css" />
</head>
```
