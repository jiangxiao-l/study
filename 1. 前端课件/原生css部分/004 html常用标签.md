# html常用标签

#### 1、无语义标签

```html
<div></div>
<span></span>
```

#### 2、常用语义标签

```html
<hn></hn> 标题
<p></p> 段落
<pre></pre> 原文本
<br /> 换行
<hr /> 分割线
```

#### 3、文本标签

```html
<i></i> 斜体字
<em></em> 斜体字，表示强调
<b></b> 粗体字
<strong></strong> 粗体字，表示强调(语气更强) 
<del></del> 删除的文本
<ins></ins> 插入的文本
<sub></sub> 下标字
<sup></sup> 上标字
<ruby>
	拼音<rt>pinyin</rt>
</ruby> 中文注音，h5新增
```

#### 4、其他标签

```html
<section></section> 块
<small></small> 小号字体
```
