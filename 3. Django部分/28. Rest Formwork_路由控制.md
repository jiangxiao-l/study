## 三、方式三

```python
全自动路由:(基本不用) 视图类必须继承：ModelViewset
    
 from rest_formework.viewsets import ModelViewset
 
#1. 生成一个router对象
from rest_framework import  routers
router = routers.DefaultRouter()

#2. 需要传两个参数，第一个参数是匹配的路径，第二个是参数，是视图类
router.register('publish',views.Publish)

# 3 .导入include模块
from django.conf.urls import url,include

urlpatterns = [
    #4.直接include
    url('', include(router.urls)),
]


// 注 url(r'a/', include(router.urls)), 相当于include后面的是二级路由，在浏览器上子先写 a/....
```
