## 二、全局配置

```python
   // 在 setting.py文件中的中间件里面配置
    
       #1.放页面的时候，应该放在最前面,实现了response  --> 放在最前面
      'django.middleware.cache.UpdateCacheMiddleware',
```

## 三、单页面缓存

```python
// 利用装饰器实现数据的缓存

import time

#1、导入相应的包
from django.views.decorators.cache import cache_page

#设置缓存的时间，单位：秒，可以实现数据缓存为5s
@cache_page(5)
def timeout(request):
    ctime = time.time()
    return  render(request,'timeout.html',locals())
```

## 三、页面局部的缓存

```html
<body>
    缓存的时间：timeout可以任意命名,可以实现数据的缓存5秒
    {% load cache %}
    {%  cache 5  'timeout' %}
    {{ ctime  }}
    {% endcache %}
<hr>
    实际时间：
    {{ ctime }}
</body>
```
